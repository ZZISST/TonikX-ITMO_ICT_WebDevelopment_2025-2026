# Tour Agency Frontend

Клиентская часть приложения туристического агентства, реализованная на React + TypeScript с использованием shadcn/ui.

## Технологии

- **React 18** — библиотека UI
- **TypeScript** — типизация
- **Vite** — сборщик
- **TanStack Query** — управление серверным состоянием
- **React Router** — маршрутизация
- **Zustand** — состояние приложения
- **shadcn/ui** — UI-компоненты
- **Tailwind CSS** — стили
- **React Hook Form + Zod** — формы и валидация
- **Axios** — HTTP-клиент

## Функциональность

### Авторизация
- Регистрация пользователя
- Вход в систему (JWT)
- Выход из системы

### Туры
- Просмотр списка туров
- Фильтрация по городу
- Детальная информация о туре
- Создание, редактирование, удаление туров

### Бронирования
- Создание бронирования
- Просмотр своих бронирований
- Отмена бронирования

### Отзывы
- Просмотр отзывов к туру
- Добавление отзыва с оценкой (1-10)

### Профиль
- Просмотр данных профиля
- Редактирование даты рождения

## Установка

```bash
# Клонирование
cd laboratory_work_4/frontend

# Установка зависимостей
npm install

# Настройка окружения
cp .env.example .env
# Отредактируйте .env, указав URL бэкенда

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

## Переменные окружения

| Переменная | Описание | Пример |
|------------|----------|--------|
| VITE_API_URL | URL бэкенда | http://localhost:8000 |

## Структура проекта

```
src/
├── api/              # API-клиенты (axios)
├── components/       # React-компоненты
│   ├── layout/       # Header, Footer, Layout
│   └── ui/           # shadcn/ui компоненты
├── pages/            # Страницы приложения
├── store/            # Zustand хранилища
├── types/            # TypeScript типы
├── lib/              # Утилиты
├── App.tsx           # Корневой компонент
├── main.tsx          # Точка входа
└── index.css         # Глобальные стили
```

## Связь с бэкендом

Работает с FastAPI бэкендом из `laboratory_work_3`. Убедитесь, что бэкенд запущен и CORS настроен.
