# Tour Agency Frontend

Клиентская часть приложения **Tour Agency** — туристического агентства с возможностью просмотра туров, бронирования и оставления отзывов.

## Технологии

- **React 18** — библиотека для создания пользовательских интерфейсов
- **TypeScript** — типизация JavaScript
- **Vite** — сборщик проекта
- **TanStack Query** — управление серверным состоянием
- **React Router** — маршрутизация
- **Zustand** — управление состоянием приложения
- **shadcn/ui** — компоненты пользовательского интерфейса
- **Tailwind CSS** — утилитарный CSS-фреймворк
- **React Hook Form + Zod** — формы и валидация
- **Axios** — HTTP-клиент

## Основные функции

### Авторизация
- Регистрация нового пользователя
- Вход в систему
- Выход из системы
- JWT-аутентификация

### Туры
- Просмотр списка туров
- Фильтрация по городу
- Детальная страница тура
- Создание нового тура (для авторизованных)
- Редактирование тура
- Удаление тура

### Бронирования
- Создание бронирования
- Просмотр своих бронирований
- Отмена бронирования

### Отзывы
- Просмотр отзывов к туру
- Добавление отзыва (для авторизованных)
- Оценка тура (от 1 до 10)

### Профиль
- Просмотр данных профиля
- Редактирование даты рождения

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

## Структура проекта

```
src/
├── api/          # API-клиенты
├── components/   # React-компоненты
│   ├── layout/   # Компоненты макета
│   └── ui/       # UI-компоненты (shadcn)
├── lib/          # Утилиты
├── pages/        # Страницы приложения
├── store/        # Zustand-хранилища
└── types/        # TypeScript-типы
```

## Связь с бэкендом

Приложение работает с FastAPI бэкендом из лабораторной работы №3. 
URL API настраивается через переменную окружения `VITE_API_URL`.
