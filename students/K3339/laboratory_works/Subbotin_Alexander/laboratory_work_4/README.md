# Лабораторная работа №4

Клиентская часть приложения туристического агентства на React + shadcn/ui.

## Структура

```
laboratory_work_4/
└── frontend/           # React приложение
    ├── src/            # Исходный код
    ├── docs/           # MkDocs документация
    └── README.md       # Инструкции
```

## Быстрый старт

1. Запустите бэкенд из `laboratory_work_3`
2. Перейдите в `frontend/`
3. Выполните:
   ```bash
   npm install
   npm run dev
   ```
4. Откройте http://localhost:3000

## Документация

```bash
cd frontend/docs
mkdocs serve
```

## Технологии

- React 18 + TypeScript
- Vite
- shadcn/ui (Radix UI + Tailwind CSS)
- TanStack Query
- React Router
- Zustand
- React Hook Form + Zod
