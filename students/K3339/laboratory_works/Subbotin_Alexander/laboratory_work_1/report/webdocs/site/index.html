Лабораторные работы по Web-программированию

О курсе

Курс включает изучение основ веб-разработки, работы с сокетами, создания веб-серверов и клиент-серверных приложений.

Структура работ

Лабораторная работа №1

Работа с сокетами
Включает в себя: - UDP клиент-сервер для обмена сообщениями - TCP клиент-сервер для вычислений - HTTP сервер с HTML страницей - Многопользовательский чат с threading - Веб-сервер с обработкой GET/POST запросов

Технологии: Python socket, threading, HTTP protocol

Лабораторная работа №2

Основы Django
Создание веб-приложения на Django Framework: - Настройка проекта и приложений Django - Модели данных и ORM - Представления (Views) и маршрутизация (URLs) - Шаблоны (Templates) и статические файлы - Администрирование Django Admin - Работа с формами Django Forms

Технологии: Django, SQLite, HTML/CSS, Bootstrap

Лабораторная работа №3

Django REST API
Разработка RESTful API с использованием Django REST Framework: - Создание API endpoints (GET, POST, PUT, DELETE) - Сериализаторы для преобразования данных - Аутентификация и авторизация API - Документация API с помощью drf-spectacular - Тестирование API endpoints - Обработка JSON данных

Технологии: Django REST Framework, JWT токены, Swagger/OpenAPI

Лабораторная работа №4

Frontend на Vue.js
Создание клиентского приложения и интеграция с API: - Настройка проекта Vue.js 3 - Компоненты и реактивность Vue - Маршрутизация с Vue Router - Состояние приложения с Pinia/Vuex - HTTP клиент для работы с API (Axios) - Адаптивный дизайн и UI компоненты

Технологии: Vue.js 3, Vue Router, Pinia, Axios, CSS/SCSS
